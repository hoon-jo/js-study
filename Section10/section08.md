# Template

## Template 처리

**템플릿 처리 방법**

` 특수 기호를 사용해서 안에 문자열을 넣고 문자열 안에 javascript를 사용할 때 \${temp}방식으로 처리한다.

ES6 이전엔 표현식을 다음과 같이 일반 문자열 안에 집어넣었다.  
ex)

```js
const str = "저는" + name + "입니다."
```

```js
const template = `저는 ${name} 입니다.`
```
